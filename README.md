# Lock In Blocker

Blue-lock themed site blocker wherein you get berated by Blue Lock characters for not locking in

## Installation

### From Source
1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions`
3. Enable **Developer mode** (toggle in top-right corner)
4. Click **Load unpacked**
5. Select the extension folder

### Manual Installation 
1. Download the latest `LockInBlocker-v1.0.0.zip` from [Releases](https://github.com/totorolivesalone/BlueLockBlocker/releases/tag/extension)
2. Extract the zip file to a folder on your computer
3. Follow steps 2-5 from "From Source" above

## Customization 
Edit `index.js` and modify the `targetSites` array:
```javascript
const targetSites = [
  "yoursite.com"
];
```
